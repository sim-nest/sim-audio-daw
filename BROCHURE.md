# sim-audio-daw

In one line: a code-first audio workstation -- shape sound, patch a live signal graph, and host real plugin formats.

## What it gives you

Ready-made sound-shaping blocks, a patch bay that connects them into one signal flow and renders it start to finish, a real-time engine that feeds a sound card without dropouts, bridges that let SIM effects appear as CLAP, LV2, VST3, or WebAssembly plugins and run those formats in return, and sound-card backends for every platform with a hardware-free lane for testing.
